Installation :
==============

After getting the sources you have to extract them thanks to the command line tar (see the help to set the right options)
When the files are extracted you have to get into the source folder (cd command):
Then do : /usr/bin/python setup.py build (to build everything)
		  export PATH=$PWD/build/scripts-2.6:$PATH (to add the scripts to the PATH)
		  export PYTHONPATH=$PWD/build/lib:$PYTHONPATH
